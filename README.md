# Django REST API Project

This is a Django project implementing a REST API for managing persons.

### Technology Used
 
- Python 3.10.12
- Django 3.2
- Rest API
- Django Rest Framework 3.14.0

### Steps to setup the project

    1) Clone the GIT repo
    2) Create virtual environment and install requirement txt file  "pip3 install -r requirement.txt"
    3) Run runserver command
    4) Go to "localhost:8000/swagger" and access the APIs
    4) For admin panel Go to "localhost:8000/admin" and use 'admin' as username and 'admin' as password.

#### NOTE: I have already attached the sqlite3 database as 'db.sqlite3' and already added dummy data in it.
